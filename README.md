# NestJS Starter Project

## Description

Starter project for [NestJS](https://docs.nestjs.com/).

## Installation

```bash
# install dependencies
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

## run unit tests in watch mode
$ npm run test:watch

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## Building the app

```bash
# production build
$ npm run build
```

## utility scripts

```bash
# formatting files
$ npm run format

# check lint
$ npm run lint
```
